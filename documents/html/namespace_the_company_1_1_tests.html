class  AddEmployeeTests
  Brief Description This class is used to test the AddEmployee methods in the Container class. The methods tested include GetFulltimeEmployeeProperties, GetParttimeEmployeeProperties, GetContractEmployeeProperties, GetSeasonalEmployeeProperties and AddEmployee. Some of the methods in this class require user input for testing (they will have a string called dataToPassIn, and will use Console.SetIn() and StringReader). To test the methods with user input, Console.SetIn() and StringReader are used to simulate the user input.

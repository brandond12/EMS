Detailed Description

Brief Description The Seasonal Employee class is used to store and manage data about a employee who is hired for a season. This class is a child to Employee class. It adds the season the employee is hired for and how much the employee is paid per item completed/created. If the constructor creates a invalid employee, a exception is thrown. All other errors result in a defined return.

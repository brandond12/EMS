Detailed Description

Brief Description This class is used to test the ParttimeEmployee methods in the AllEmployee class. The methods tested include the constructors, Details, SetDateOfBirth (all 3 overloaded methods), SetDateofHire (all 3 overloaded methods), SetDateofTermination(all 3 overloaded methods), SetHourlyRate and ToString.

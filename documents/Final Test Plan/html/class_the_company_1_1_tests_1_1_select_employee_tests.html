TheCompany.Tests.SelectEmployeeTests Class Reference

Brief Description This class is used to test the SelectEmployee methods in the Container class. The methods tested include SelectEmployee, IsThisTheDesiredEmployee and DisplayEmployeeDetails. Some of the methods in this class require user input for testing (they will have a string called dataToPassIn, and will use Console.SetIn() and StringReader). To test the methods with user input, Console.SetIn() and StringReader are used to simulate the user input.
